# 🧠 Nutrition and Child Growth Analysis (Synthetic Public Health Project)

## 📌 Project Overview
This project simulates a public health data science pipeline focused on analyzing how nutrition, socioeconomic conditions, and health factors influence child growth outcomes such as stunting, wasting, and underweight status.

A large-scale synthetic dataset (10,000 records) is generated to model relationships between nutrition intake, household conditions, disease exposure, and growth outcomes.

## 🎯 Objectives
- Understand how nutrition impacts child growth
- Identify key malnutrition risk factors
- Simulate public health surveillance data
- Build predictive machine learning models
- Visualize regional malnutrition risk using GIS methods

## 📊 Dataset
Includes demographics, nutrition indicators, health variables, and growth outcomes such as height, weight, BMI, stunting, wasting, and underweight status.

## ⚙️ Methodology
1. Synthetic data generation with realistic correlations  
2. Exploratory Data Analysis (EDA)  
3. Machine learning classification (Random Forest)  
4. GIS-style heatmap visualization of malnutrition risk  

## 🧠 Key Insights
- Rural regions show higher malnutrition risk  
- Low protein/calorie intake strongly affects growth  
- Poor sanitation increases disease exposure  
- Dietary diversity is highly protective
